Zebra.Sdk.Card.Graphics.Barcode Namespace

Provides interfaces for card printer barcode generation.

Classes

ZebraBarcodeFactory Factory used to acquire an instance of a font specific barcode object.

Interfaces

BarcodeUtil Base interface used for drawing a barcode.
Code128Util Interface used to configure and draw a Code128 barcode.
Code39Util Interface used to configure and draw a Code39 barcode.
CodeEANUtil Interface used to configure and draw an EAN barcode.
CodePDF417Util Interface used to configure and draw a PDF417 barcode.
QRCodeUtil Interface used to configure and draw a QRCode barcode.
