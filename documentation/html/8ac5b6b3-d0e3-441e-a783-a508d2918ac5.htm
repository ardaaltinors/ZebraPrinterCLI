ImportSampleCode

Follow these instructions in order to import the Windows Card Desktop Demos into Visual Studio.

Importing Example Code into Visual Studio

  1. Navigate to [Platform]/[build version]/demos/visualStudio within your Link-OS Multiplatform SDK installation (Where [Platform] is the desired platform and [build version] corresponds to the desired build).

  2. Right-click DeveloperDemo_Card_Desktop.zip and select Extract All... to unzip the example code.

  3. From the Visual Studio menu bar, select File > Open > Project/Solutions....

  4. Navigate to the DeveloperDemo_Card_Desktop folder that you unzipped in step 2.

  5. Select DeveloperDemo_Card_Desktop.csproj.

  6. Click Open.
