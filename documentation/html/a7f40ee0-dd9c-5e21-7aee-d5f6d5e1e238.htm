ProfileGetPrinterDownloadableObjectFromPrinter Method

Retrieves a file from the profile and returns the contents of that file as a byte array including all necessary file wrappers for re-downloading to a Zebra printer.

Remarks

Will retrieve the following file extensions: ZPL, GRF, DAT, BAS, FMT, PNG, LBL, PCX, BMP, WML, CSV, HTM, BAE, TXT
Files transferred between different printer models may not be compatible.

Exceptions

ZebraIllegalArgumentException If the filePath is invalid, or if the file does not exist on the printer.
