The Link-OS Multiplatform SDK Command Line Tool

The ZebraCard Multiplatform SDK command line tool can be used to perform various printer operations from the command line. It can be used to get printer status, update firmware, print, discover usb and network printers, and perform various other operations.

Steps to Use the ZebraCard Multiplatform SDK Command Line Tool to Perform Printer Operations

  1. Open a command prompt

  2. Navigate to "PC.NET/[build version]/lib" within your Link-OS Multiplatform SDK installation directory (where [build version] corresponds to the desired build)

  3. Enter ZSDK_API_CARD_DESKTOP.exe help

  4. Follow the instructions in the help
